Información sobre el Sensor de Posición del Cigüeñal (CKP)

¿Qué es un Sensor de Posición del Cigüeñal (CKP)?

El Sensor de Posición del Cigüeñal (CKP) es un dispositivo que detecta la posición y la velocidad de giro del cigüeñal del motor. Esta información es fundamental para que la computadora del vehículo controle la inyección de combustible y el encendido, asegurando un funcionamiento eficiente y sincronizado del motor.

Cambio y mantenimiento de un Sensor de Posición del Cigüeñal (CKP)

El sensor CKP debe revisarse si el motor presenta fallas como dificultad para arrancar, apagones inesperados o pérdida de potencia. Se recomienda inspeccionar visualmente el sensor y su conector, limpiar cualquier suciedad o residuo, y reemplazarlo si está dañado o arroja códigos de error. Es importante utilizar repuestos de calidad y seguir las indicaciones del fabricante para su instalación.

¿Qué código marca en el escáner el Sensor de Posición del Cigüeñal (CKP)?

Cuando el sensor CKP presenta fallas, la computadora del vehículo puede registrar códigos de diagnóstico (DTC) como los siguientes:

  • P0335: Mal funcionamiento del circuito del sensor CKP.
  • P0336: Rango/rendimiento del circuito del sensor CKP.
  • P0337: Entrada baja en el circuito del sensor CKP.
  • P0338: Entrada alta en el circuito del sensor CKP.
  • P0339: Circuito intermitente del sensor CKP.

Estos códigos pueden variar según el modelo y año del vehículo. Es recomendable consultar el manual de servicio y realizar un diagnóstico adecuado.
