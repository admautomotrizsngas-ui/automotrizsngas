Información sobre el Sensor de Posición del Árbol de Levas (CMP)

¿Qué es un sensor de posición del árbol de levas (CMP)?

El sensor de posición del árbol de levas (CMP) es un dispositivo que detecta la posición exacta del árbol de levas y envía esta información a la computadora del vehículo. Esto permite sincronizar la inyección de combustible y el encendido, optimizando el rendimiento y la eficiencia del motor.

Cambio y mantenimiento de un sensor CMP

El sensor CMP debe revisarse si el motor presenta fallas de encendido, dificultad para arrancar o pérdida de potencia. Se recomienda inspeccionar y limpiar el sensor y sus conexiones eléctricas periódicamente. Si el sensor está dañado o arroja códigos de error, debe ser reemplazado siguiendo las indicaciones del fabricante.

¿Qué código marca en el escáner el sensor CMP?

Cuando el sensor CMP presenta fallas, la computadora del vehículo puede registrar códigos de diagnóstico (DTC) como los siguientes:

  • P0340: Mal funcionamiento del circuito del sensor de posición del árbol de levas.
  • P0341: Rango o funcionamiento incorrecto del sensor de posición del árbol de levas.
  • P0342: Entrada baja en el circuito del sensor de posición del árbol de levas.
  • P0343: Entrada alta en el circuito del sensor de posición del árbol de levas.

Estos códigos pueden variar según el modelo y año del vehículo. Es recomendable consultar el manual de servicio y realizar un diagnóstico adecuado.
