Información sobre el Sensor de Detonación

¿Qué es un sensor de detonación?

El sensor de detonación es un dispositivo que detecta las vibraciones anómalas producidas por la detonación o "pistoneo" en el motor. Envía señales a la computadora del vehículo para ajustar el avance del encendido y evitar daños en el motor.

Cambio y mantenimiento de un sensor de detonación

El sensor de detonación debe revisarse si el motor presenta ruidos metálicos, pérdida de potencia o códigos de error relacionados. Se recomienda inspeccionar y limpiar el sensor y sus conexiones eléctricas periódicamente. Si el sensor está dañado, debe ser reemplazado siguiendo las indicaciones del fabricante.

¿Qué código marca en el escáner el sensor de detonación?

Cuando el sensor de detonación presenta fallas, la computadora del vehículo puede registrar códigos de diagnóstico (DTC) como los siguientes:

  • P0325: Mal funcionamiento del circuito del sensor de detonación.
  • P0326: Rango o funcionamiento incorrecto del sensor de detonación.
  • P0327: Entrada baja en el circuito del sensor de detonación.
  • P0328: Entrada alta en el circuito del sensor de detonación.

Estos códigos pueden variar según el modelo y año del vehículo. Es recomendable consultar el manual de servicio y realizar un diagnóstico adecuado.
