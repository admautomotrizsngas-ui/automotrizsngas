Información sobre el Sensor TPS

¿Qué es un sensor TPS?

El sensor TPS (Throttle Position Sensor) es un dispositivo que mide la posición de la mariposa de aceleración en el motor. Esta información es enviada a la computadora del vehículo para ajustar la inyección de combustible y el avance de encendido, permitiendo un funcionamiento eficiente y una respuesta adecuada del motor al acelerar.

Cambio y mantenimiento de un sensor TPS

El sensor TPS debe revisarse si el motor presenta fallas como aceleración irregular, pérdida de potencia o dificultad para mantener el ralentí. Se recomienda limpiar periódicamente el sensor y sus conexiones eléctricas. Si el sensor está dañado o arroja códigos de error, debe ser reemplazado siguiendo las indicaciones del fabricante y utilizando repuestos de calidad.

¿Qué código marca en el escáner el sensor TPS?

Cuando el sensor TPS presenta fallas, la computadora del vehículo puede registrar códigos de diagnóstico (DTC) como los siguientes:

  • P0120: Mal funcionamiento del circuito del sensor TPS.
  • P0121: Rango/rendimiento del circuito del sensor TPS.
  • P0122: Entrada baja en el circuito del sensor TPS.
  • P0123: Entrada alta en el circuito del sensor TPS.
  • P0124: Circuito intermitente del sensor TPS.

Estos códigos pueden variar según el modelo y año del vehículo. Es recomendable consultar el manual de servicio y realizar un diagnóstico adecuado.
